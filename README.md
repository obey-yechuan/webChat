# webchat
Vue + Express项目框架
![项目构架](/src/assets/构建.png)
## Project setup
> 安装`vue`项目所需的包和server文件夹下`node`所需的包
```
npm install
cd server 
npm install
```

### Compiles and hot-reloads for development
> 启动服务器
```
npm run server
```
> 新开` crtl + ````` `终端启动vue项目
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
